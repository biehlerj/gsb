# gnome-settings-backup

CLI for importing and exporting GNOME settings. Written in Go.

## Why use this

Are you tired of trying to set up GNOME the way you want it when you are setting up a new machine? Well now you can import and export your GNOME settings using this command line tool that is an extension/improvement on this [Perl script found on AskUbuntu](https://askubuntu.com/a/217310)
